# teamsclone

## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```

## Tech Stack:

 - node.js
 - WebRTC using regular javascript APIs
 - PeerJS
 - socket.io library
 - EJS
 - CSS
 - Heroku for hosting website
